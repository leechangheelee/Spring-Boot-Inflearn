## **스프링 부트 활용**
  * 테스트 (Junit4)
    * 샘플 코드
      ```java
      /* sample/SampleController.java */
      ...
      @RestController
      public class SampleController {

          @Autowired
          private SampleService sampleService;

          @GetMapping("/hello")
          public String hello() {
              return "hello " + sampleService.getName();
          }
      }
      ```
      ```java
      /* sample/SampleService.java */
      ...
      @Service
      public class SampleService {

          public String getName() {
              return "changhee";
          }
      }
      ```
